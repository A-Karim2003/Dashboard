# Dashboard Project

A React dashboard application built to explore and experiment with the Shadcn UI library and its theming capabilities.

## Purpose

This project serves as a playground for testing Shadcn UI components and theming features. The focus is on the UI/UX implementation rather than the actual data displayed.

## Tech Stack

- **React 19** - UI framework
- **Vite** - Build tool
- **Shadcn UI** - Component library
- **Tailwind CSS 4** - Styling
- **React Router 7** - Navigation
- **Recharts** - Data visualization
- **TanStack Table** - Data tables

## Features

- Dark/Light theme toggle with system preference support
- Collapsible sidebar with persistent state
- Responsive grid layouts for different screen sizes
- Multiple dashboard views (Homepage, User, Users)
- Charts and data visualizations
- Data tables with sorting and filtering
